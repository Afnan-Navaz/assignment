# shareText
MERN web app to share text based posts
### Requirements
* Node.js  
* Yarn
* mongoDB
### Local Setup Instruction
1. `git clone https://github.com/Afnan-Navaz/assignment.git`
2. `cd assignment`
3. **Back-End**
	* (localhost:5000)
	* `cd backend	`
	* `sudo service mongod start`
	* `yarn`
	* `yarn start`
4. **Front-End**
	* (localhost:3000)
	*  `cd frontend`
	* `yarn`
	* `yarn start`
