# Assignment
MERN web app to share text based posts
## Requirements
* Node.js  
* Yarn
* mongoDB
## Local Setup Instruction
1. `git clone https://github.com/Afnan-Navaz/assignment.git`
2. `cd assignment`
3. **Back-End**
	* `cd backend	`
	* `sudo service mongod start`
	* `yarn`
	* `yarn start`
4. **Front-End**
	*  `cd frontend`
	* `yarn`
	* `yarn start`
